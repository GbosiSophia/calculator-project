# Requirements

## Project Name
Sophia's Calculatpr

## Description
A simple web calculator built with HTML, CSS and JavaScript.

## Functional Requirements
- User can click number buttons to build an expression that would show on the display box.
- User can use +, -, x, /, ^ operations
-User can clear (C)
- User can calculate result (=)
- Prevent multiple decimals in a number

## Non-Functional Requirements
- Simple UI, easy to use
- Works on Chrome/Edge
- Responsive enough for laptop and phone screens

##  Tools Used
- HTML, CSS, JavaScript
